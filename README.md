# hotel_revenue insights Dashboard 
Hotel Revenue Insights | Power BI Project
Analyzed large-scale hotel revenue data spanning three months to uncover key business trends and performance metrics.

Built interactive dashboards using Power BI to present complex data in a clear, visually intuitive, and decision-focused manner.

Utilized DAX (Data Analysis Expressions) to calculate critical business measures and perform statistical analysis.

Defined and implemented core KPIs, including Total Revenue Realized, Occupancy Percentage, and Average Customer Rating, to evaluate operational efficiency.

Calculated Occupancy Percentage to measure room utilization based on successful bookings and availability
